## 3. 校验规则可配置 `scripts/validate.py`

### 已实现

- `-i / --input`：指定输入 XML，默认 `test_data/error/naxml-posjournal 10477 035502 1.xml`
- `-o / --output-id`：指定输出 id（如 `10477 035502 1`），不指定则从 `-i` 的文件名解析

### 使用示例

```bash
# 默认（等同原行为）
python scripts/validate.py

# 指定输入
python scripts/validate.py -i "test_data/error/naxml-posjournal 10477 035502 1.xml"

# 指定输入 + 输出 id（若解析规则不适用）
python scripts/validate.py -i "path/to/other.xml" -o "custom-id"
```
